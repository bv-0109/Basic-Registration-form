📄 Registration Form using Flask

📌 Project Overview:

This project is a simple Flask web application that provides:

A Registration Page where users can enter their details (Name, City, Phone Number, Email Address).
A Confirmation Page that displays the submitted information in a clean, vertical layout.
The project demonstrates basic Flask routing, HTML form handling, and template rendering.

⚙️ Requirements:

Make sure you have the following installed:
Python 3.8+
Flask (Python web framework)

▶️ Running the Project:

Open your browser and go to:

http://127.0.0.1:5000/

Fill out the registration form and click Register.
You will be redirected to the confirmation page showing your submitted details.

📂 Project Structure:

Code:

project-folder/
│
├── new.py                # Main Flask application
├── templates/
│   ├── register.html      # Registration form page
│   └── confirmation.html  # Confirmation output page
